# 縁陣（Enjin）LP

## プロジェクト概要

**縁陣（Enjin）** の採用支援サービス向けランディングページです。  
「出会いが採用を変える。」をコンセプトに、採用課題を抱える中堅・中小企業の人事担当者・経営者をターゲットに構成されています。

---

## 実装済み機能

### ページ構成（セクション）
| セクション | 内容 |
|-----------|------|
| SEC01 Hero | キャッチコピー・CTAボタン・パーティクル背景 |
| SEC02 採用の現実 | 数値アニメーション付き離職率・コスト計算 |
| SEC03 問題の本質 | 3つの穴カード・悪循環ダイアグラム・突破口 |
| SEC04 サービス詳細 | タブ切替（Encounter / Immerse / Transform） |
| SEC05 導入効果 | Before/After比較カード |
| SEC06 根拠・実績 | 研究知見・設計者コメント・先行モニターCTA |
| SEC07 差別化 | 競合との違い3点 |
| SEC08 チーム | 林 宣伶・大内 結貴プロフィール |
| SEC09 料金・FAQ | 料金表・アコーディオンFAQ |
| SEC10 最終CTA | 無料相談予約・サービス資料ダウンロード |

### インタラクション
- **ナビゲーション**: スクロール時にグラスモーフィズム効果
- **スクロールリビール**: 各セクションの要素が画面内に入ると`fade-in + slide-up`
- **カウンターアニメーション**: 離職率「33.8%」がスクロールで数値カウントアップ
- **サービスタブ**: タブクリックでコンテンツ切替（フェードイン）
- **FAQアコーディオン**: クリックで展開/折りたたみ
- **フローティングCTA**: ヒーロー通過後〜CTAセクション手前で表示
- **Canvasパーティクル**: ヒーローセクションに浮遊ドット演出
- **サイクルダイアグラム**: 悪循環図の各アイテムを順番ハイライト
- **スムーススクロール**: アンカーリンクのスムース移動

### デザイン特徴
- ダークテーマ（深夜ブルー系）
- Noto Sans JP / Noto Serif JP
- Font Awesome 6.4.0 アイコン
- CSS Custom Properties（変数）で統一カラー管理
- フルレスポンシブ対応（モバイル〜デスクトップ）
